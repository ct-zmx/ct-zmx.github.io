## 0x01 表格  

表格的基本格式如下:
```a
|   表头   |  表头   |
|   ----   |  ----   |
| 单元格1  | 单元格2 |
| 单元格3  | 单元格4 |
```
其中，`|`用来分隔不同的单元格，用 `-`分隔表头和其他行。效果如下：

|   表头   |  表头   |
|   ----   |  ----   |
| 单元格1  | 单元格2 |
| 单元格3  | 单元格4 |

同时可以设置表格的对其方式：
>`-:`  标题和内容右对齐  
>`:-`  标题和内容左对齐  
>`:-:` 标题和内容居中对齐  

示例如下：
```
|   表头   |  表头   |  表头    |
|   ----:  |  :----  | :----:  |
| 右对齐格  | 左对齐格 |  居中格 |

```
效果如下：
|   表头   |  表头   |  表头   |
|   ----:  |  :----  | :----:  |
| 右对齐格  | 左对齐格 |  居中格 |

（默认为居中对齐）

## 0x02 链接
链接的形式两种：
```
[链接名称](链接地址)  

<链接地址>
```
举个例子：
```
[百度](https://www.baidu.com/)

<https://www.baidu.com/>
```
效果如下：  
[百度](https://www.baidu.com/)

<https://www.baidu.com/>

---
还可以有更简洁的方法，有点类似于论文中的引用：
```
这是我博客的地址[ctamx][1]

[1]:https://ctzmx.cool/
```
效果如下：

这是我博客的地址[ctamx][1]

[1]:https://ctzmx.cool/

其中，地址可以在文章末尾再设置（即为`[1]:https://ctzmx.cool/` 可以在文章最后给出）
